# sales-prediction
# Sales Prediction using Python

## Introduction
This project focuses on predicting sales using advertising data. The analysis is performed using Python and Machine Learning techniques.

## Dataset
Dataset Used: Advertising.csv

### Features
- TV Advertising Budget
- Radio Advertising Budget
- Newspaper Advertising Budget
- Sales

## Tools and Libraries Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Objectives
- Analyze advertising data.
- Perform data cleaning and exploration.
- Visualize relationships between advertising channels and sales.
- Train a regression model for sales prediction.
- Evaluate model performance.
- Analyze the impact of advertising on sales.

## Project Workflow
1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Feature Selection
6. Model Training
7. Sales Prediction
8. Model Evaluation
9. Business Insights

## Results
The Linear Regression model was used to predict sales based on advertising expenditure. The model demonstrated a strong relationship between advertising investment and sales performance.

## Conclusion
Sales can be effectively predicted using advertising data. The project highlights the importance of data analysis and machine learning in business decision-making.
